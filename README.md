# 回音笺 HuiYinJian

把你正在做的事，翻译成可见回报。

## 项目简介

“回音笺（HuiYinJian）”是一个 AI 驱动的反馈系统，面向那些持续投入时间与精力、却难以清晰看见阶段性成果的人。很多真实发生的成长都停留在日常叙述里，例如“我最近做了不少事情”“我学到了很多”“这段经历以后可能有用”，但这些内容往往没有被进一步结构化、表达化和资产化。

回音笺要解决的问题，不是提醒用户完成任务，而是把原本隐性的过程价值显化出来。系统接收用户通过文字或语音描述的自然语言输入，将其转译为结构化的“可见回报”，帮助用户看见自己做了什么、产出了什么、能力如何提升、哪些内容可以进入简历，以及下一步如何让回报更清晰、更可展示。

## MVP 目标

第一版最小可行产品聚焦于一条最短闭环：

1. 接收用户的一段自然语言输入。
2. 对输入进行基础结构化分析。
3. 输出一份标准化反馈结果。
4. 为后续接入模型调用、前端展示和语音输入预留接口结构。

## 当前规划功能列表

- 文本输入分析接口
- 将自然语言转为结构化反馈对象
- 输出任务摘要、产出、能力、证据、简历表述建议
- 生成当前事项的回报评分与原因说明
- 提供“如何让回报更可见”的下一步建议
- 为未来语音转写接入保留扩展空间
- 为前端展示和后续存储设计统一输出格式
